HTML

HyperText Markup Language, is the standard markup language used to create and design documents on the World Wide Web. It provides the structure and layout for web pages by using a variety of elements and tags.

File Format

.html eg. index.html

Comment

A comment is a piece of text that is not displayed in the web browser but is embedded in the HTML code to provide notes or explanations about the code.

They can also be used to temporarily disable code without deleting it.

HTML Tags

HTML tags are the building blocks of HTML (HyperText Markup Language) documents. They are used to create and define the structure and content of web pages. An HTML tag consists of the tag name enclosed in angle brackets (< >).

Singular Tags

These tags do not wrap around content and do not need a closing tag. Examples include <img>, <br>, <hr>, and <meta>.

Paired Tags

These tags wrap around content and require both an opening and a closing tag. Examples include <p>, <a>, <div>, and <span>.

<p> </p> A paragraph tag

Defines a paragraph.

<h1> </h1> Defines a top-level heading.

We can use <h1> </h1>, <h2> </h2>, <h3> </h3>, <h4> </h4>, <h5> </h5>

div container <div> </div>

It is a versatile container element used to group and style sections of content. It does not inherently have any visual representation, but it can be styled using CSS (Cascading Style Sheets) and is often used to apply layout and design to sections of a web page.

image tag <img src="" alt="">

It is used to embed images in a web page. It is a self-closing (singular) tag, meaning it does not have a closing tag.

The src and alt(Alternative text for the image.) are attributes for img tag.

Using the alt attribute correctly ensures that images are accessible, improve SEO, and provide a better user experience for everyone.
